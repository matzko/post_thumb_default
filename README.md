== Description ==

This WordPress plugin defaults the post thumbnail to the first attachment.

== Installation ==
------------------

1. Upload `post_thumb_default` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
